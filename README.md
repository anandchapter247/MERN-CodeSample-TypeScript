# Sample Code

## Frontend:

```
React + Redux + Redux-Logic + TypeScript
```

## Backend

```
NodeJS + ExpressJS + TypeScript with Webpack
```

## Availible Commands

### Frontend:

#### Start dev server

```
yarn start
```

or

```
npm start
```

#### Create Build:

```
yarn build
```

or

```
npm run build
```

### Backend

#### Run dev server (need to run two commands)

```
npm run dev:watch
```

and then in other terminal

```
npm run dev
```

#### Create Build

```
npm run build
```

#### Build and Run Build

```
npm run prod
```

#### Generate API Doc

```
npm run gen:doc
```

```
npm run dev:ts
```

#### Create build with ts compiler

```
npm run tsc
```
